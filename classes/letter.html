class Letter(symbol: Symbol, exponent: Exponent)¶

This class represents a letter.

Note

Public access of both instance variables (sym and exp) is encouraged.

Parameters:

  • symbol (Symbol) – The symbol of the letter. All strings are valid symbols.

  • exponent (Exponent) – The exponent of the letter.

>>> Letter("a", 1)
a
>>> Letter("x", 3)
x³
sym: Symbol¶
exp: Exponent¶
__eq__(other: object) → bool¶

Checks for equality between two letters. Two letters are equal \(\iff\) the symbol and the exponent are equal. Returns NotImplemented if other is not a Letter.

Parameters:

other (object) – The letter to compare against.

>>> Letter("a", 4) == Letter("a", 4)
True
>>> Letter("a", 2) == Letter("b", 3)
False
__lt__(other: object) → bool¶

Checks if self < other. The following ordering is followed:

\[a < a^2 < \ldots < a^{-2} < a^{-1} < b < \ldots\]
Parameters:

other (object) – The object to compare against.

Returns NotImplemented if other is not a Letter.
